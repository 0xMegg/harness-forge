## Lessons Learned
- `run-harvest.sh`의 `--target` 파라미터 누락 버그 발견 — config.json의 measurement.command를 사용하도록 수정 필요
- 점수 delta가 0인 이유: harness-report는 파일 존재/개수 기반 — 기존 파일 내 규칙 추가는 점수에 반영되지 않음. 규칙 품질 점수 항목 추가 고려
- Dedup이 정상 동작하여 1차 수집 항목 재수집 방지 확인
- Double-gating Gate 2 흐름 (stash → apply → measure → revert → pop) 정상 검증
- Concreteness gate가 추상적 제안 4개를 효과적으로 걸러냄
